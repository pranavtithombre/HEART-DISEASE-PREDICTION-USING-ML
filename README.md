# HEART-DISEASE-PREDICTION-USING-ML

## Project Overview

This project predicts the likelihood of heart disease using Machine Learning (Logistic Regression) on the Heart Disease UCI Dataset.

## Features

- Data preprocessing
- Feature scaling
- Logistic Regression model
- User input prediction
- Model evaluation using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report

## Dataset

Heart Disease UCI Dataset

Features:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Rest ECG
- Maximum Heart Rate
- Exercise Induced Angina

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
